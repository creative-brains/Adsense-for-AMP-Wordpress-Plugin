# Adsense for AMP Wordpress Plugin

## Overview

Wordpress plugin to setup Google Adsense in AMP pages.

## Demo

[Click here to see plugin in action](https://adsenseforamp.com/demo/)

## Key features
* Google adsense support on all AMP articles.
* Support for Responsive and fixed size ads.
* Ads optimized for all sizes of mobile devices.
* Support to add custom banner ads.
* Add google adsense link ads on AMP pages.
* 'Above the Post Content' ad placement support.
* 'Below the Post Content' ad placement support.
* Fully validated with AMP validator.	

![Above the content ads in AMP](/images/above-the-content-responsive.png)

## Installation

The “Adsense for AMP” plugin requires “AMP” plugin for WordPress by “Automattic”.  So if you haven’t installed AMP plugin yet, kindly follow these steps:

1. Log in to your WordPress admin and navigate to Plugins > Add new.
2. Inside “Search Plugins” field, type “AMP”. From the results, install the AMP plugin as found in this URL: https://wordpress.org/plugins/amp/
3. After installation, activate the plugin via Plugins > Installed Plugins > AMP
Once that is done, to install the “Adsense for AMP Pro” plugin, follow these steps:

1. Log in to your WordPress admin and navigate to Plugins > Add new.
2. Inside “Search Plugins” field, type “Adsense for AMP”.
3. After installation, activate the plugin via Plugins > Installed Plugins > Adsense for AMP
Once the plugin is activated, you can navigate to the settings page for “Adsense for AMP”.
